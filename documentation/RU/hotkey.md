# Горячие клавиши

На macOS используйте `Cmd` вместо `Ctrl` для стандартных комбинаций!

## Глобальные

| Горячая клавиша | Действие |
|---|---|
| Ctrl+O | Файл: Открыть файл |
| Ctrl+Shift+O | Файл: Открыть папку |
| Ctrl+S | Файл: Сохранить |
| Ctrl+Shift+S | Файл: Сохранить как |
| Ctrl+, | Файл: Настройки |
| Ctrl+W | Файл: Закрыть вкладку редактора |
| Ctrl+Shift+W | Файл: Закрыть рабочее пространство |
| Ctrl+F | Правка: Найти в редакторе |
| Ctrl+` | Вид: Показать/скрыть терминал |
| Ctrl+B | Вид: Показать/скрыть файловое дерево |

## Редактор кода

| Горячая клавиша | Действие |
|---|---|
| Ctrl+F | Открыть строку поиска |
| Esc | Закрыть строку поиска |
| Ctrl+Plus / Ctrl+= | Увеличить масштаб |
| Ctrl+Minus | Уменьшить масштаб |
| Ctrl+Scroll | Масштаб колёсиком мыши |
| Ctrl+Space | Вызов автодополнения |
| ALT+MWheel UP/DOWN | Переключение между вкладками |
| ALT+Arrow LEFT/RIGHT | Переключение между вкладками |
| Ctrl+W | Закрыть текущую вкладку |
| Mouse Middle Button | Закрыть текущую вкладку |
| Tab | Вставить пробелы (или таб — по настройкам) |
| Shift+Tab | Уменьшить отступ |
| Enter | Авто-отступ (включая `{}`) |

## Терминал

| Горячая клавиша | Действие |
|---|---|
| Ctrl+C | Копировать выделение (прерывание НЕ отправляет) |
| Ctrl+V | Вставить в терминал |
| Ctrl+A | Выделить текущий набранный ввод (или всё, если ввода нет) |
| Ctrl+Break | Отправить сигнал прерывания запущенному процессу |
| Enter | Выполнить команду |
| Up / Down | История команд |

## HEX-редактор

| Горячая клавиша | Действие |
|---|---|
| Left / Right | Переместить курсор |
| Shift+Left / Shift+Right | Выделить по символу |
| Up / Down | Переместить по строке |
| Shift+Up / Shift+Down | Выделить по строке |
| PageUp / PageDown | Страница вверх/вниз |
| Shift+PageUp / Shift+PageDown | Выделить по странице |
| Home / End | Начало/конец строки |
| Shift+Home / Shift+End | Выделить до начала/конца строки |
| Ctrl+Home / Ctrl+End | Начало/конец документа |
| Ctrl+Shift+Home / Ctrl+Shift+End | Выделить до начала/конца документа |
| Ctrl+A | Выделить всё |
| Ctrl+Z | Отменить |
| Ctrl+Y / Ctrl+Shift+Z | Повторить |
| Ctrl+X | Вырезать |
| Ctrl+C | Копировать |
| Ctrl+V | Вставить |
| Insert | Переключить режим вставки/замены |
| Backspace | Удалить предыдущий байт |
| Delete | Удалить текущий байт |
